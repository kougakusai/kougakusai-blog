上記のように表示させるためには以下のようにコーディングする必要があります。

```md
# h1

## h2

### h3

> ブロッククオート

**太字**

普通の文字
```

このように簡単な構文を使って文字に効果を持たせることが出来ます。基本的な Markdown 記法です。

他の構文も存在するので良かったら<a href="https://mdxjs.com/getting-started#table-of-components">こちら</a>をご覧ください。

これは FrontMatter という記法です。

<div class="filename">

つまりファイルを作って FrontMatter で書きさえすればページが作れる。後は夢中になって内容書くだけ。

このように表示したいコードを逆クオート 3 つで囲むことで表示できます。今回は拡張子 go を指定してみます。

```go
package main
import "fmt"
func main() {
    fmt.Println("Hello, World!")
}
```
